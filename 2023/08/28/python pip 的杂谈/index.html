我被 pip 整的头痛死了。
最初接触 python 的时候，就因为环境变量栽过坑，虽然是小问题，自己查资料就弄好了。后来又被 pip 的各种无法解释的报错弄的头大。例如有：
超长问题报错，找不到各种文件，带????的报错……
问过 ChatGPT，虽然看起来挺专业的，实际上也未能解决问题。
有一说一，它最后那句“考虑重新安装 Python，确保按照官方指导进行操作”可能是最有用的。（笑）

如果你和我一样，遇到网络上搜不到的、无法解释的、内容很长的报错，可以尝试重装了。

Python 重装指南：

建议按照下面来操作：

  1. 测试各个版本的 pip 是否可用。
    按 Windows 键，搜索 python 并右键“打开文件所在位置”，找出电脑里各个版本的安装目录。
    打开该目录下Scripts目录，用命令行测试是否能使用 pip。
    如果都不可以用，进行步骤 2。

  2. 卸载电脑里的 python

    注意！！

    默认 Python launcher 打开了你最新版本的解释器。但用python 3.X即可指定版本，如果嫌麻烦设置，在重装前，建议删除所有版本的 python，以免安装旧版本与新版本的冲突。

    在控制面板里逐一删除各版本 python，“Python launcher”可以不删。
    另外，建议查看环境变量里是否还有遗留。

  3. 重装 python
    镜像站的速度很快，建议找到一个没有安装过的版本下载。
    点进需要的版本后一般很多文件，找到 .exe 后缀的即可，例如python-3.9.4.exe。
    个人推荐 3.8.6（玄学）当然需要新版本的可以无视。

    如果上述操作无误，打开命令行后用pip --vision或安装某个库后，无报错就是成功🌶️！
