欢迎来到“老猫的 Blog”的介绍页！这里介绍了网站的来源以及未来发展。语言有点啰嗦，挑感兴趣的看吧。

01-本网站的来源


1. 我和网站的一些事

最初我接触网站的搭建，是因为我的朋友们和我组建了一个叫 TXC 的团队（名字来源三人姓氏首字母）。想做一个里面存放着我们的作品、有趣的聊天、团队的历史等内容的网页。

也就是这样，自学了一点 html, css, js，走了不少弯路，也跳过不少坑，留下一些黑历史，至今也还是半桶水（或者更少）。也勉勉强强把网页搭起来了。 TXC 团队主页目前的网址是 txc.oldcatmiku.top。

后来因为团队主页实在没有维护和更新，就连每年 35 软妹币的域名费用也懒得交，干脆给自己搭建了博客，把团队主页挪到本博客下面的子域名里去了。

2. 博客的来源

实话说来，我原本没有博客的写作习惯的，后来才发现也许这样的模式比较适合我。

有时候，在生活中度过某段经历，或是读完了某本书，或是看了一部电影，玩通了一个游戏等等，都挺想要记录下来，或者分享出去，以免自己忘记，或者让别人看见产生一点点价值。

不过我发现，要是以“发动态（或者说朋友圈）”和直接发给好友的形式，对我来说分别有两个问题：

  1. 不愿意去“灌水”，不想让一些不重要的、个人的信息充斥别人的动态（或者说朋友圈）。
  2. 发给别人，但他们看到这条消息的反应与我想象的不一致，甚至跟我的看法不一致，有落差感。

……所以，干脆就发到自己的博客上罢！呼呼呼，当然我的意思不指会在这里灌水或发牢骚！这个博客的作用将会在下面提到。

02-这里会做什么？


根据不同的 Tag ，会写不同类型的东西。一般来说，有下面几种，或分享快乐，或展示思想，或交流学习：

  • 在这里记录生活中有意思的事情：有趣的事、绷不住的话语、难忘的经历，以及更多……
  • 分享我的思想和观点：也许是玩通了某个游戏、看了某部电影的想法，也许是对生活里的问题的看法，讨论一些问题，谈人生、世界观、梦想、价值……
  • 记录一瞬间产生的有趣的脑洞，某些奇妙的世界设定。
  • 分享一些技术、照片、游戏、好听的歌，还有更多……
  • 占个位，不仅仅是上面这些。

03-我会怎样管理这个博客？


国内 or 国外？

这个网站的域名注册，服务器都架设在国外，只有 DNS 的解析在国内（目前准备调整）。
可能会问为什么要弄到国外呢？仅仅是觉得能 少一些麻烦 而已。感兴趣可以去看这个 [还没有写，别点]。

文章的管理

原则一：不会删除博客文章；
原则二：文章在主题不变的情况下，可能会被修改。

补充：【对于原则一】如果是冲动发的博客文章且 几乎没有价值觉得很社死，有几率被隐藏（可以去找到鸭）。

广告的问题

虽然我缺钱，但日流量只有这么一点……广告还是算了。嗯嗯……如果很多人看？ 呐呐~~ 反正网页托管用的是 github 的，也不花钱，广告还是不会有的（应该不会打脸）。
也可以拿钱砸我！！捐赠应该就在导航栏里面，手机的话请点右上角三条线。

排版和语言风格

如果把表达意思相近的几行文字称作一段，每一段之后会刻意加入空行。这么做为了看起来松散一些，在网页上的文字这样做也许可以提高可读性，但是缺点是读一点文字就要滑动屏幕。

写完会读一下是否有语病，也许一会儿找不出来……
要是像日常杂谈的话我会用这种类似聊天的语言写文章吧，如果是技术分享会尽量用简明的语言描述。
